Drop the evidence documents for a case here (FIR copy, witness statements,
charge-sheet, medical/forensic reports, etc.). Supported formats:

- .pdf (text-based or scanned — scanned PDFs are auto-OCR'd)
- .docx
- .png / .jpg / .jpeg / .tif / .bmp (photographed evidence)

Then run:

    uv run analyze_case.py Evidence/*.pdf

See ../README.md for the full pipeline explanation.
